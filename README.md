# Humidity Lab

短小交互式网页，用于模拟并可视化室内/室外温度、相对湿度、绝对湿度、露点以及窗玻璃结露/结霜情况。

- 主文件：`index.html`（单文件实现交互、计算与可视化）。
- 资源：图片存放在 `images/`。

快速上手
- 在浏览器中打开 `index.html`（双击或使用本地静态服务器）。
